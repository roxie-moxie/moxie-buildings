# Moxie Buildings

Daily apartment availability scraper & webapp for 400+ managed rental buildings in Downtown Chicago.

**Owner:** Moxie Real Estate Team
**Purpose:** Replace the manual daily process of pulling pricing from individual building websites. Scrape unit-level availability data (unit #, rent, sqft, availability date) and serve it through a webapp where agents can filter, sort, and generate client-ready lists.
